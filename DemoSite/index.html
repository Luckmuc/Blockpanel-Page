Installation Tutorial

  1. Clone repository:
    git clone https://github.com/Luckmuc/Blockpanel.git
  2. Change directory:
    cd Blockpanel
  3. Start with Docker:
    docker-compose up --build
  4. Visit in browser:
    http://localhost:1105
